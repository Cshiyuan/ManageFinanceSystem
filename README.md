# ManageFinanceSystem

**家庭财务管理系统（课程大作业）**

基于 MFC 的家庭财务管理系统，支持多用户登录、收支记录管理和数据持久化存储。包含控制台版本（C++）和 Windows 窗体版本（VB.NET）两种实现。

## 功能

- 用户注册与登录
- 收支条目的增删改查
- 数据本地持久化
- 多成员管理

## 项目结构

```
├── MFS/        C++ 控制台版本 (Visual C++)
└── F_M_MFC/    VB.NET Windows 窗体版本
```

## 技术栈

- **语言**: C++ / VB.NET
- **框架**: MFC (C++ 版) / Windows Forms (VB.NET 版)
- **IDE**: Visual Studio
